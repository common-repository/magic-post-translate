Automatic Translate Posts & Pages with Deepl

== Description ==
Automatic Translate Posts & Pages with <a href="https://www.deepl.com/">Deepl</a>.

You need a Deepl API key to apply translations.

You can :

* Set your Deepl API key
* Choose betweeen 9 languages : English, German, French, Spanish, Portuguese, Italian, Dutch, Polish, Russian
* Translate chosen posts
* Translate chosen pages
* Translate chosen custom post type
* Translate all posts in a category


<a href="https://www.deepl.com/privacy.html">Privacy Terms of Deepl</a>

== Installation ==
1. Activate the plugin
2. Go to Settings > Magic Post Translate
3. Configure your settings and Deepl API
4. Go into your posts or pages, choose which one you want to translate and choose "Generate translation with Deepl" and apply
